# Sumdinger
This game was built in swift and is similar to Yahtzy

<img width="377" height="754" alt="Screenshot 2026-04-09 at 10 31 39 AM" src="https://github.com/user-attachments/assets/05321cbc-58d7-4a0e-9e25-985f1f25e4ae" />

The goal is to get the sum of two dice to fill out all the numbers 3-11\
Also if there isn't two dice that add up to any of the numbers you can select either double 6s or double 1s to reroll\
Once all the sums of two dice are filled out the game ends
